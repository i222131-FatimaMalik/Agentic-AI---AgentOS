1 Context and Motivation
Modern AI systems in 2026 are no longer single-prompt chatbots. They are autonomous agentic
systems capable of decomposing goals, reasoning across multiple steps, interacting with unreliable
tools, verifying uncertain information, repairing failures, and resisting adversarial manipulation.
Such systems are now deployed in:
•software engineering copilots
•cybersecurity incident response
•financial auditing and compliance
•operations planning and logistics
•scientific experimentation assistants
In all real deployments:
•single-shot generation is unreliable
•verification is mandatory
•tool misuse is dangerous
•adversarial data is common
•failure recovery is essential
This assignment trains you to engineer such an autonomous reasoning system explicitly.
2 High-Level Objective
You must build a complete agentic AI framework in pure Python that:

5 Formal Action Protocol
At every reasoning step exactly one JSON object must be produced.
Tool call:
{" type ":" tool "," name ":" calc "," args ":{" expression ":"3*(4+5) "}}
Final answer:
{" type ":" final "," answer ":"27" ," confidence ":0.95}
Any malformed JSON or schema violation must trigger reflection repair.
